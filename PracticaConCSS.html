Elementos HTML Más Usados

  • <html> - Etiqueta raiz
  • <head> - Contiene metadatos, enlaces a hojas de estilo, scripts, etc.
  • <body> - Contiene el contenido visible de la página web.
  • <div> - Contenedor genérico
  • <h1 'a' h6> - Tipos de encabezados
  • <p> - Párrafo
  • <a> - Enlace
  • <img> - Imagen
  • <table> - Tabla
  • <ul> - Lista no ordenada
  • <ol> - Lista ordenada
  • <li> - Define un elemento dentro de una lista
  • <nav> - Define una seccion de navegacion

Propiedades CSS Más Usadas

  • color - Color del texto
  • background-color - Color de fondo
  • margin - Margen exterior
  • padding - Relleno interior
  • border - Borde
  • font-family - Define las fuentes para el texto
  • font-size - Define el tamaño de la fuente
  • width - Define el ancho de un elemento
  • height - Define la altura de un elemento
